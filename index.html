Who speaks what on GitHub?

These visualization were built for the GitHub Data Challenge 2014. The source can be found on GitHub

These three visualizations provide insight into the language skills of users on GitHub. The three distinct techniques are displayed below. Want to know more? Read more here on how I built this!
